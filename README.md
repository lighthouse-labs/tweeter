# Tweeter Project

Tweeter is a simple, single-page Twitter clone.

 HTML, CSS, JS, jQuery and AJAX front-end skills, and their Node, Express back-end skills.

## Getting Started

npm start to start the server

## Dependencies

- Express
- Node 5.10.x or above
- body-parser
- chance
